# Automated Web Testing Suite

A Python-based web automation testing project built using Selenium and PyTest.

## Features
- Automated login testing
- Add-to-cart workflow validation
- Navigation testing
- Page Object Model (POM) architecture
- PyTest-based execution

## Tech Stack
- Python
- Selenium
- PyTest
- WebDriver Manager

## Project Structure
automation-testing-suite/
│
├── tests/
├── pages/
├── utils/

## Run Tests

pytest -v
